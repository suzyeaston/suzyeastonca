# Claude Code Guide for suzyeastonca

## Copy Voice

Write all site copy in Suzy's voice: punchy, dry, confident, a little weird. Punk-bassist-turned-builder energy.

- State things like they're obvious. Never explain or justify why something is cool.
- Short sentences. Lowercase-casual is fine. Specific beats generic.
- Banned words: seamless, innovative, cutting-edge, leverage, solutions-driven, "you can actually."
- If a line sounds like an ad, kill it.

Good: "Official status pages lie. This one doesn't." / "Tools that shouldn't work but do."
Bad: "A browser-based Vancouver you can actually navigate." (explainy, ad-like)

Apply to: all page copy, card descriptions, CTAs, section headers, meta descriptions.
